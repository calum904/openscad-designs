# Blue Light Block Red Light Panel

This is a fixed stand for the MID [Blue Light Block Red Light Panel](https://www.blockbluelight.co.uk/collections/red-light-therapy-panels/products/red-light-therapy-powerpanel-mid) (540mmx210mmx65mm).

It raises it some 10cm off the ground, which is ideal for being able to sit on the floor and have the panel against your front/back.
